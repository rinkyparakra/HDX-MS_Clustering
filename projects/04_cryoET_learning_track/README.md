# Cryo-ET Learning Track

This folder documents my transition into computational cryo-electron tomography and subtomogram averaging.

## Goal

Build practical familiarity with cryo-ET computational workflows relevant to high-resolution structure determination.

## Topics

- Tilt-series alignment
- CTF estimation and correction
- Tomogram reconstruction
- Particle picking in tomograms
- Subtomogram averaging
- Resolution estimation
- Hybrid structural analysis with complementary methods such as HDX-MS

## Software to Explore

- RELION
- Dynamo
- EMAN2
- IMOD
- Warp / M
- cryoSPARC tools
- ChimeraX
- Python-based image-analysis workflows

## Planned Practice Project

Use a public cryo-ET tutorial dataset to document:

1. Raw tilt-series organization
2. Alignment strategy
3. Reconstruction parameters
4. Subtomogram extraction
5. Initial averaging
6. Biological interpretation

## Why this connects to my background

My HDX-MS work focuses on dynamic and heterogeneous protein complexes. Cryo-ET and subtomogram averaging provide a complementary way to study macromolecular assemblies in more native-like environments. This learning track reflects my goal of combining structural proteomics, cryo-EM/cryo-ET, and computational analysis.
